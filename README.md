🚀 About The Project

Buddy is an AI-powered role-based voice interview platform that simulates real-time job interviews using Large Language Models.

It dynamically generates interview questions based on selected roles and evaluates voice responses intelligently — helping candidates improve confidence and communication before real interviews.

✨ Features

🚀 Fully Voice-Based Interview System
🧠 Role-Based Question Generation
🔄 Real-Time AI Follow-Up Questions
📊 Intelligent Response Evaluation
💾 Candidate Data Storage
⚡ Powered by Groq LLM
