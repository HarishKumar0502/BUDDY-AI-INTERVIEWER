🚀 About The Project

Buddy is an AI-powered role-based voice interview platform built using FastAPI and Groq LLM.

It simulates real-time job interviews through voice interaction, dynamically generates technical questions based on selected roles, and evaluates responses intelligently.

Designed to help candidates improve communication skills, confidence, and technical clarity before real interviews.

✨ Core Features

🎤 100% Voice-Based Interview System.
🧠 Role-Based Question Generation
🔄 Real-Time AI Follow-Up Questions
📊 Intelligent Response Evaluation
⚡ High-Performance Backend with FastAPI
💾 Candidate Data Storage (CSV)
